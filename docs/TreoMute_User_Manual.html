What is TreoMute?

Mute Alert Sounds for a Predetermined Time Interval.

Downloads

The latest version of TreoMute can be downloaded here: (PRC | ZIP). The latest version of this user manual can be viewed online here.

Installing TreoMute

HotSync or copy TreoMute.PRC to your Treo phone. It will appear in the Treo Launcher. It should be installed in the internal memory for proper functioning.

Uninstalling TreoMute

Disable TreoMute and delete from the [Home]->[Menu]->[Delete] window.

Compatibility

TreoMute will work only on all current Treo series Smartphones. It has been developed on a Cingular Treo 650 running the latest updater available on the Palm website.

Operation

Invoking TreoMute from the Preferences Panel will display the TreoMute Preferences form. The options are self-explanatory.

More details will be made available in due course. Only one thing to remember, some applications may trap the activation keys before TreoMute can receive the notifications, in which case, TreoMute will not be activated. However, there will be no problem using TreoMute in most Palm Treo native applications.

Features of TreoMute

  • Small PRC (~ 12 KB).
  • No hook-ups to 'hede' notifications, though it uses 'hvch' notifications.
